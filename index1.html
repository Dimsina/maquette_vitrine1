My_meetic

Le but du projet était de créer un site de rencontres à la manière d'un "Meetic". Nous avions comme obligations de programmer en PHP orienté objet.

Il nous était fortement conseiller d'utiliser le pattern MVC ainsi que de vérifier les données envoyées avant de traiter les réponses.

Comme restrictions il nous était interdit d'utiliser des plugins JQuery ainsi que d'utiliser les scripts Javascript de framework type Bootstrap
